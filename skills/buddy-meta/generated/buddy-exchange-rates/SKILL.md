# Buddy Exchange Rates Skill

## Призначення
Отримання актуальних курсів валют **USD** та **EUR** від API ПриватБанку.

## Коли використовувати
- Користувач запитує курс долара або євро
- Потрібно показати поточні курси купівлі/продажу
- Інформація про обмін валют

## Як викликати

```bash
python3 exchange_rates.py
```

## Аргументи
- Без аргументів — повертає курси USD та EUR

## Відповідь

**Успіх:**
```json
{
  "status": "success",
  "source": "ПриватБанк",
  "rates": {
    "USD": {"currency": "Долар США", "ccy": "USD", "buy": 41.50, "sale": 42.10},
    "EUR": {"currency": "Євро", "ccy": "EUR", "buy": 44.80, "sale": 45.50}
  },
  "formatted": "💱 Курси валют від ПриватБанку..."
}
```

**Помилка:**
```json
{
  "status": "error",
  "error": "Помилка з'єднання",
  "message": "Не вдалося отримати курси валют"
}
```
